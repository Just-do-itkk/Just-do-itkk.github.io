智能车PCB（一）——初识PCB

Posted on 2020-01-21 Edited on 2020-01-24 In 智能车 Views:

智能车自制PCB是一件非常有成就感的事情。但是这其中所包含的知识真的很多，记得19年上半年我磨了一个月才真正意义上的画出了可以用的板子（也仅仅只是可以用而已）。接触新鲜的事物是一件非常容易地事情，但是熟练地掌握、融会贯通确实需要大量地实践地，不过好在我非常幸运地有车队地学长帮助：画板子前常常“骚扰”昂总，画板子时参考蒋总的博客，疯狂刷AD教程，每回画好板子都是位总、蒋总、昂总三连发，然后再接着修修补补，真的是超级艰辛。

现在虽说算不上是有多精通PCB绘制，但是基本的操作也算是得心应手了，最后的感觉是画智能车的PCB其实应该算是比较简单的，用昂总的话说是“连连看”也没太大问题哈哈哈，当然这也是句玩笑话，高性能的板子对于整个智能车系统来说还是很有助益的，合理的布局有益于在安装各类传感器时保证车有一个较好的结构（尤其是直立车）；良好的集成也可以防止智能车最后变成“盘丝洞”。

其实现在回顾起来，对于一个弱电专业的学生，在有一定专业课基础之后，使用AD画一块智能车的板子并不是太难的事情；最大的困难是在上手的最初，你需要了解一些最基本的知识，这也是正是我想写下来的东西。

Altium Designer

所谓”工欲善其事，必先利其器”。首先来介绍PCB绘制的工具之所以会使用Altium Designer，首先是当时正在上的电子工艺实践有要求对给出的电路利用AD布线，其次是学长们都在用，理所当然的就一直用下去了。所以接下来所有的关于PCB设计的讨论都是基于Altium Designer展开的。

Altium Designer 是原Protel软件开发商Altium公司推出的一体化的电子产品开发系统，主要运行在Windows操作系统。这套软件通过把原理图设计、电路仿真、PCB绘制编辑、拓扑逻辑自动布线、信号完整性分析和设计输出等技术的完美融合，为设计者提供了全新的设计解决方案，使设计者可以轻松进行设计，熟练使用这一软件使电路设计的质量和效率大大提高。—-摘自百度百科

PCB绘制

PCB介绍

PCB（Printed Circuit Board），中文名为印制电路板，其实就是电子元器件的支撑体，作为电子元器件电气连接的载体组成特定的电路模块。现在一般电路板直接可以发给PCB厂家（嘉利创、捷配等）制作，制作时间只需两到三天，价格也非常的便宜，所以已经很少有人用三氯化铁腐蚀电路板了。

使用AD绘制PCB

绘制前

使用AD绘制PCB之前，有很多的东西需要准备好。

１、设计好的电路。（对应原理图的绘制）

​ 电路的设计十分的重要，原理图直接决定了整个电路是否可以正常地工作，尤其是绘制主板时，单片机的引脚分配决定了整个嵌入式系统是否能真正的正常工作。建议在绘制好原理图后进行反复的检查，确认无误后再进行PCB布线工作。

２、电路涉及的元件原理图库。（原理图库可以自行绘制，也可寻找现成的）

​ 电路的原理图库是用在绘制原理图时的，绘制原理图时，使用每个元器件对应的原理图来进行电气连接，所以原理图最关键的是引脚个数一定要与实际元件对应。

​ 其次，一般元件的原理图库和封装库是意义对应的，直观的说就是原理图的引脚和对应封装的引脚应该要一一对应，否则会造成最终的PCB电气连接发生错误，板子就废了。

３、电路涉及的元件封装库。（封装库可以根据所需元件的数据手册给出的封装尺寸自行绘制，不过有很多现成的封装库，可以通过各种途径获取）

​ 元件封装库其实就是最后在PCB板上看到的用来放置元器件的过孔或者金属表面，所以要保证它与元件的引脚对应，在绘制好PCB之后购买元器件时实际是依据PCB导出的元件列表来进行的。

绘制流程

常规情况下绘制PCB的大致流程如下：

1、新建PCB工程文件
并向工程文件里添加PCB文件和原理图文件
2、元件库、封装库设计
部分元器件厂商或者经销商不提供元件库和封装库，只给了元器件尺寸图，所以需要自行设计元件库文件或是封装库文件
元件库设计
新建 .SchLib 文件：File-New-Library-Schematic Library}
使用Place下拉菜单或使用快捷工具栏放置图形，引脚等，记得保存！
封装库的设计
新建.PcbLib文件：File-New-Library-PCB Library
使用Place下拉菜单或使用快捷工具栏，画线，放置孔位，记得保存！
3、原理图的绘制
(1)新建.SchDoc文件 ：File-New-Schematic
(2)添加元器件库和封装库（在软件底部菜单栏System中勾选Libraries，打开侧面工具栏，在侧面工具栏中点击Libraries…>Add Library找到自己保存的库文件并添加）
(3)添加各元器件（可直接从侧面工具栏中选择元器件拖入原理图中，在拖动过程中按Tab键修改元器件信息）
(4)添加网络标识（Place-Net Label 快捷键PN）
(5)确定各元器件的封装（打开封装管理器Tools-Footprint-Manager快捷键TG，可挨个修改、检查各元器件封装）
4、PCB的绘制
(1)导入原理图中的各元器件：Design-Import-Changes-From…
(2)元器件布局(元器件布局原则：靠近边缘摆放，由边缘向中心布局,按原理图分块布局,元器件是否能成功安装，考虑元器件大小，高度等等 )
(3)连线(连线原则：线不走直角，标签不压线,VCC加粗，GND不连，通过铺铜实现等）
(4)完成布局连线后就是铺铜操作Place-Polygon… 或者点击快捷工具栏中的Place Polygon Plane按钮,在弹框中的Net Options中的Connect to Net选项选GND，勾选去除死铜Remove Dead Copper
(5)铺铜完成后，可在Design -> Rules…中修改或增加相应规则
如：VCC加粗，元器件间距等等
5、规则检查
修改规则后点击Tools -> Design-Rule-Check -> Run Design…
进行规则检查，并修改对应的错误和警告，完成PCB的绘制。
至此，这块PCB板子就可以投给厂家生产了。

小结

PCB绘制的过程中存在很多的细节，这些将在在之后进行详细的讨论，但是要明确的是整个绘制的过程的走向，对于入门者来说，明白自己正在做什么，下一步要做什么是很重要的。
